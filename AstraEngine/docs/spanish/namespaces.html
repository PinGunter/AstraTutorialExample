AstraCore
Astra Core is a C++ library to develop visualization apps in Vulkan using rasterization or hardware-accelerated ray-tracing.

 CRendererClase renderer. Contiene todo lo necesario para renderizar la escena. Se encarga de todo el proceso de dibujado. Contiene también los descriptor sets y pipeline de post procesado. Es una clase que interactua con vulkan a bajo nivel y no está pensada para ser sobreescrita.
La forma en la que funciona el renderer es con dos render pass. El primero rasteriza o traza con rayos la escena y guarda el resultado en una textura. Esa textura luego se utiliza en el segundo render pass para mostrarse en pantalla junto con la interfaz de ImGui. De esta forma podemos combinar el trazado de rayos para modelos de la escena con la rasterización para fuentes y elementos gráficos de la interfaz
 CSceneClase Escena. Contiene las mallas, instancias, luces y cámara. Esta es para rasterización
 CSceneRTClase que agrega los elementos necesarios para el raytracing a una escena
